Connecting the nodes

The idea in this prototype is to show all cities/towns initially. A user can click on a node to see the relevant information in that place. A user can also connect one node to another to specify the path travelled by holding on the CTRL or COMMAND key and dragging from one node to another. Relevant data can be retrieved upon user interactions. As with prototype 1, the prototype 2 can also be used to record the straight line diagram.
